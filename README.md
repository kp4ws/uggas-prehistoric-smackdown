# 🦖 Ugga's Prehistoric Smackdown

> **Status:** 🕹️ Prototype

A 2D side-scrolling platformer set in prehistoric times featuring beautiful pixel art environments and classic, nostalgic platforming gameplay. 

**Inspired by:** *Banjo-Kazooie, Mario, Brother Bear (GBA), and Genndy Tartakovsky's Primal TV series.*

## 🎮 Gameplay & Features
- **Classic Platforming:** Step into the prehistoric era as Ugga and navigate through multiple dangerous, handcrafted levels.
- **Power-Ups:** Collect unique items and power-ups hidden across the world to enhance Ugga's abilities.
- **Exploration:** Focuses on fun, rewarding exploration tied to tight platforming mechanics.

## 🛠️ Tech Stack
- **Game Engine:** Godot Engine
- **Languages:** 
  - `GDScript`
